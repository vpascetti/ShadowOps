## Core Features (Live Demo in 15 Minutes)

### 1. Executive Briefing Dashboard
**For: Plant Managers, COO, CEO**

One screen shows:
- Total jobs in plant
- How many are late, at risk, on track
- Top 10 critical alerts
- Which work centers are bottlenecks

**Value:** Skip the morning report meeting. Know plant health in 30 seconds.

### 2. Smart Alerts
**For: Production Supervisors, Planners**

Automatic detection of:
- Late jobs (already past due)
- At-risk jobs (schedule vs. progress gap >25%)
- Projected late jobs (AI prediction based on current pace)
- Scrap spikes, cycle time variance, idle time excess

**Value:** Fix problems proactively instead of firefighting reactively.

### 3. Priority Run Lists
**For: Work Center Operators, Supervisors**

Each work center gets a prioritized queue based on:
- Due dates
- Current progress
- Risk scores
- Downstream dependencies

**Value:** Eliminate "what should I run next?" decisions. Reduce late jobs by 40%.

### 4. Time-Travel Analysis
**For: CI Teams, Root Cause Analysis**

Rewind plant state to any past date:
- "What did the plant look like 2 weeks ago?"
- "Why was Job X late?"
- "How would delaying this order impact everything?"

**Value:** Learn from history. Plan scenarios. Justify decisions with data.
